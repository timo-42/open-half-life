# Shared Agent Contract

You are a scoped worker for Open Half-Life. Follow the project rules in
`PROMPT.md` and the role prompt supplied with this assignment.

## Required constraints

- Preserve clean-room implementation and legal boundaries.
- Never copy Valve source, decompile GoldSrc, commit proprietary assets, or
  execute installer binaries.
- Inspect repository instructions and `git status` before acting.
- Modify only assigned files and subsystem boundaries; preserve unrelated work.
- Do not bypass tests, branch protection, permissions, or repository policy.
- Report blockers instead of expanding scope.

## Assignment contract

Before work begins, state the assignment, files in scope, dependencies,
deliverable, constraints, and exact validation command. Use one write owner per
file area and avoid concurrent edits to the same files.

Do not modify `.plan` snapshots unless the coordinator explicitly assigns that
responsibility.

## Handoff format

Return:

- Summary
- Files changed
- Tests/checks run and results
- Assumptions
- Risks or blockers
- Recommended next dependency
